## Product

### Create
* Method:  `POST`
* URL:  `/products`
* header: token
* body: `name=[string]*`,` stock=[number]*` ,`price=[number*]`, `image=[linkimage]`
* Output
    * 201: 
    ``` 
        {
            _id: '1',
            name: 'Kaos oblong',
            stock: '10',
            price: '20000',
            image: 'url....'
            seller: {
                _id: UserId,
                fullname: 'lorem lorem'
            },
            created_at: '...',
            updated_at: '...'
        }
    ```
    * 400:
    ```
        {
          message: 'Name is required'
        }
    ```
    * 409
    ```
      {
          message: 'Name has been used'
      }
    ```

### Update product
* Method:  `PUT`
* URL:  `/products/:_id`
* header: token
* body: `name=[string]*`,` stock=[number]*` ,`price=[number]*`, `categories=[arrayOfCategoryId]`
* Output
    * 200: 
    ``` 
        {
            _id: '1',
            name: 'Kaos oblong',
            stock: '10',
            price: '20000',
            image: 'url....'
            seller: {
                _id: UserId,
                fullname: 'lorem lorem'
            },
            created_at: '',
            updated_at: ''
        }
    ```
    * 400:
    ```
        {
            message: 'Name is required'
        }
    ```
    * 409
    ```
        {
            message: 'Name has been used'
        }
    ```
    * 404:
    ```
        {
            message: 'id not found'
        }
 
    ```

### Delete product
* Method:  `DELETE`
* header: token
* URL:  `/products/:_id`
* Output
    * 200: 
    ``` 
        {
            "n": 1,
            "ok": 1,
            "deletedCount": 1
        }
    ```
