# e-commerce

**User: Register**
----
  Register new user to app.

* **URL**

  /register

* **Method:**

  `POST`
  
*  **URL Params**

   **Required:**
 
   None

* **Data Params**

   name: [String],
   email: [String],
   password: [String]

* **Success Response:**

  * **Code:** 201 <br />
    **Content:** <br />```{
        "_id": "5e243323b1155a4fa129cb8c",
        "name": "Peter",
        "email": "peter@gmail.com"
    }```
 
* **Error Response:**

  * **Code:** 400 <br />
    **Content:** <br />`{
        "error": [
            "Please fill a valid email address"
        ]
    }`
    
    **Content:** <br />`{
        "error": [
            "Email already exist"
        ]
    }`
